# Cadastro de Pessoas - CRUD Fullstack

Aplicação CRUD completa para cadastro de pessoas com **NestJS** (backend) e **Vue.js 3** (frontend), ambos com **TypeScript**.

## Como rodar

### Backend

```bash
cd backend
npm install
npm run start:dev
```

Roda em `http://localhost:3000`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Roda em `http://localhost:5173`.
